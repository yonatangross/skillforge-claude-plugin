## Formula

```
Raw = Impact*0.25 + (10-Effort)*0.20 + (10-Risk)*0.20 + Align*0.20 + Innov*0.15
Final = Raw * (0.70 if critical concerns, 0.85 if 3+ high concerns, else 1.0)
```
