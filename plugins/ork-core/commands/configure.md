# OrchestKit Configuration

Load and follow the skill instructions from the `skills/configure/SKILL.md` file.

Execute the `/ork:configure` workflow to:
1. Choose configuration preset (Complete, Standard, Lite, Hooks-only)
2. Customize skill categories (AI/ML, Backend, Frontend, Testing, Security, DevOps)
3. Toggle agents on/off (Product, Technical)
4. Configure hooks (Safety, Productivity, Quality Gates, Notifications)
5. Enable MCP integrations (context7, sequential-thinking, memory, playwright)
6. Configure CC 2.1.7 settings (turn duration, MCP deferral threshold)
7. Preview and save configuration

## Arguments
- No arguments: Start interactive configuration wizard
