Complete narration script templates for different video types with timing annotations and TTS formatting.

## Format-Specific Adjustments

### Vertical (9:16) Narration

```
Adjustments:
- Shorter sentences (visual space limited)
- More pauses (eyes jump between text zones)
- Key words at start of sentences (text often overlaps)
- Simpler vocabulary (casual viewing context)

Example adjustment:
Standard:  "With this feature, you can automatically generate reports."
Vertical:  "Auto-generate reports. One click. Done."
```

### Square (1:1) Narration

```
Adjustments:
- Moderate pacing (professional context)
- Complete sentences (LinkedIn, Instagram feed)
- Clear topic transitions (less visual real estate)
- Professional tone

Example adjustment:
Standard:  "Check this out! Super cool feature incoming."
Square:    "Here's a feature that changes workflows. Let me show you."
```

### Horizontal (16:9) Narration

```
Adjustments:
- Can be more detailed (more screen space)
- Longer phrases acceptable
- Room for visual demonstrations
- Can reference on-screen elements

Example:
"On the left, you'll see the input panel. On the right, real-time results."
```
