# Platform Pacing Rules

Detailed pacing specifications for each major video platform.

## TikTok

### Algorithm Factors
- **Watch time percentage** is the primary ranking signal
- **Loop completions** boost distribution significantly
- **Early engagement** (likes, comments, shares in first hour) critical

### Pacing Requirements
```
Duration Sweet Spots:
├── 7-15 seconds   → Highest completion rate
├── 21-34 seconds  → Optimal for storytelling
├── 45-60 seconds  → Educational content
└── 60-180 seconds → Deep dives (lower reach, higher engagement)

Cut Frequency:
├── Minimum: 1 cut every 3 seconds
├── Optimal: 1 cut every 1.5-2 seconds
└── Maximum: Don't exceed 3 cuts per second (seizure risk)

Hook Requirements:
├── 0-0.3s: Movement or pattern interrupt
├── 0.3-0.5s: Face or human element
└── 0.5-1.0s: Value proposition visible
```

### Visual Velocity
```
Frame    Purpose              Duration    Elements
────────────────────────────────────────────────────
1        Hook                 0.3-0.5s    Face + motion + text
2        Context              0.5-1.0s    Setup the premise
3-10     Content              1.5-2.5s ea  Rapid delivery
11-15    Payoff               1.0-2.0s ea  Satisfy the hook
Final    Loop setup           0.5-1.0s    Connect to frame 1
```

### Sound Strategy
```
Music BPM Range: 120-150 (matches natural scroll rhythm)
Voice Pace: 150-180 words per minute
Sound Effects: Every 2-4 seconds
Audio Hook: Within first 0.3 seconds
```

## Instagram Reels

### Algorithm Factors
- **Saves** weighted heavily (indicates value)
- **Shares to stories** amplifies reach
- **Non-follower reach** based on early performance

### Pacing Requirements
```
Duration Sweet Spots:
├── 7-15 seconds   → Highest shares
├── 30-45 seconds  → Optimal engagement
├── 60-90 seconds  → Tutorial/educational
└── 90 seconds max → Current platform limit

Cut Frequency:
├── Minimum: 1 cut every 4 seconds
├── Optimal: 1 cut every 2-3 seconds
└── Aesthetic content: Can go slower (3-5 seconds)

Hook Requirements:
├── 0-0.5s: Visual interest
├── 0.5-1.0s: Clear subject
└── 1.0-2.0s: Value or entertainment promise
```

### Aesthetic Considerations
Instagram users expect higher production value:
```
Production Element    TikTok    Reels
───────────────────────────────────────
Raw/authentic         Preferred  Acceptable
Color graded          Optional   Expected
Smooth transitions    Optional   Preferred
Text design           Basic      Polished
```

## YouTube Shorts

### Algorithm Factors
- **Swipe-away rate** in first 3 seconds critical
- **Subscriber conversion** from Shorts to long-form
- **Watch time** (absolute, not just percentage)

### Pacing Requirements
```
Duration Sweet Spots:
├── 15-30 seconds  → Highest completion
├── 30-45 seconds  → Balanced reach/engagement
├── 45-60 seconds  → Maximum allowed
└── Note: Can't exceed 60 seconds

Cut Frequency:
├── Minimum: 1 cut every 5 seconds
├── Optimal: 1 cut every 2-4 seconds
└── Educational: Can sustain 4-6 seconds per shot

Hook Requirements:
├── 0-1s: Stop the scroll
├── 1-2s: Establish context
└── 2-3s: Promise value
```

### YouTube-Specific Patterns
```
Pattern: Subscribe Trigger
├── 0-3s: Hook
├── 3-45s: Deliver massive value
├── 45-55s: Tease related content
└── 55-60s: "Subscribe for more [specific topic]"

Pattern: Funnel to Long-Form
├── 0-3s: Intriguing question
├── 3-50s: Partial answer (value)
├── 50-60s: "Full breakdown linked above"
```

## YouTube Long-Form

### Algorithm Factors
- **Average view duration** primary ranking signal
- **Click-through rate** from impressions
- **Session time** (keeps viewers on platform)

### Pacing Requirements
```
Duration Sweet Spots:
├── 8-12 minutes   → Standard monetization
├── 12-20 minutes  → Optimal for most niches
├── 20-40 minutes  → Podcasts, deep dives
└── 40+ minutes    → Documentary, entertainment

Cut Frequency by Section:
├── Intro (0-30s): 10-15 cuts/minute (high energy)
├── Hook (30s-2m): 8-12 cuts/minute
├── Body: 6-10 cuts/minute
├── Climax: 12-20 cuts/minute
└── Outro: 4-8 cuts/minute
```

### Retention Curve Optimization
```
Time        Expected Retention    Action if Below
─────────────────────────────────────────────────
0-30s       90-95%               Strengthen hook
30s-2m      75-85%               Add pattern interrupts
2-5m        60-75%               Restate value
5-10m       50-65%               Introduce new element
10m+        40-55%               Maintain variety

Danger Zones:
├── 30-second drop: Hook isn't landing
├── 2-minute drop: Promise not being delivered
└── Mid-video drop: Pacing too slow or content weak
```

### Chapter Pacing
```
Chapter Structure for 15-minute video:
├── Cold Open: 0:00-0:30 (30s)     - Hook
├── Intro: 0:30-1:30 (60s)         - Context
├── Chapter 1: 1:30-4:30 (3m)      - First major point
├── Chapter 2: 4:30-8:00 (3.5m)    - Second point
├── Chapter 3: 8:00-11:30 (3.5m)   - Third point
├── Synthesis: 11:30-13:30 (2m)    - Bring together
└── CTA/Outro: 13:30-15:00 (1.5m)  - Action
```

## LinkedIn

### Algorithm Factors
- **Dwell time** on post (including video)
- **Professional relevance** to viewer's industry
- **Comment quality** weighted over quantity

### Pacing Requirements
```
Duration Sweet Spots:
├── 30-60 seconds  → Highest engagement
├── 60-90 seconds  → Thought leadership
├── 90-180 seconds → Detailed insights
└── 180+ seconds   → Significant reach penalty

Cut Frequency:
├── Minimum: 1 cut every 10 seconds
├── Optimal: 1 cut every 6-8 seconds
└── Can sustain longer shots for authority

Hook Requirements:
├── 0-1s: Professional context
├── 1-2s: Contrarian or valuable insight
└── 2-3s: Why it matters to their career
```

### LinkedIn-Specific Patterns
```
Pattern: Thought Leadership
├── 0-3s: Counter-intuitive statement
├── 3-30s: Evidence/reasoning
├── 30-50s: Actionable framework
└── 50-60s: Open question for comments

Pattern: Quick Win
├── 0-3s: "Here's a tip that [benefit]"
├── 3-45s: Demonstrate the tip
├── 45-60s: "Try this and let me know"
```

### Professional Pacing Considerations
```
Element              Consumer Platforms    LinkedIn
────────────────────────────────────────────────────────
Speaking pace        Fast (150-180 wpm)   Moderate (130-150 wpm)
Text on screen       Frequent, brief      Fewer, substantial
Jump cuts            Very common          Use sparingly
B-roll               Entertainment        Professional/relevant
Music                Trending audio       Subtle/optional
```

## Twitter/X

### Algorithm Factors
- **Engagement rate** in first 30 minutes
- **Quote tweets** weighted heavily
- **Video completion** for autoplay optimization

### Pacing Requirements
```
Duration Sweet Spots:
├── 15-30 seconds  → Optimal for reach
├── 30-60 seconds  → Balanced engagement
├── 60-140 seconds → Maximum without upload limit
└── Note: 2:20 limit for most accounts

Cut Frequency:
├── Minimum: 1 cut every 4 seconds
├── Optimal: 1 cut every 2-3 seconds
└── News/commentary: Can be slower

Hook Requirements:
├── 0-0.5s: Visual hook (autoplay without sound)
├── 0.5-1.5s: Context
└── 1.5-3s: Value clear even without audio
```

### Silent Viewing Optimization
Twitter videos often viewed without sound:
```
Element                  Priority
────────────────────────────────
Captions                 Required
Visual storytelling      Critical
Text overlays            Common
Sound-dependent content  Avoid
```

## Platform-Specific Templates

### TikTok 30-Second Template
```
0:00-0:01  → Hook (visual + text + audio)
0:01-0:05  → Context setup (2 cuts)
0:05-0:15  → Main content (5-6 cuts)
0:15-0:25  → Payoff/proof (4-5 cuts)
0:25-0:30  → CTA + loop setup (2 cuts)
Total cuts: 14-15 (28-30 cuts/minute)
```

### YouTube Long-Form 10-Minute Template
```
0:00-0:30  → Cold open hook (10-15 cuts/min)
0:30-1:30  → Intro + promise (8-10 cuts/min)
1:30-3:00  → First section (8 cuts/min)
3:00-4:30  → Second section (6-8 cuts/min)
4:30-6:00  → Third section (6-8 cuts/min)
6:00-7:30  → Fourth section (8 cuts/min)
7:30-9:00  → Synthesis (8-10 cuts/min)
9:00-10:00 → CTA + outro (6 cuts/min)
Average: ~8 cuts/minute = 80 total cuts
```

### LinkedIn 60-Second Template
```
0:00-0:03  → Hook statement (1-2 cuts)
0:03-0:15  → Context/problem (2 cuts)
0:15-0:40  → Insight/solution (4 cuts)
0:40-0:55  → Proof/example (2 cuts)
0:55-1:00  → CTA for comments (1 cut)
Total cuts: 10-11 (10 cuts/minute)
```
