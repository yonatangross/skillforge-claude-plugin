## Voice Settings Deep Dive

### Parameters Explained

```typescript
interface VoiceSettings {
  stability: number;        // 0.0 - 1.0
  similarity_boost: number; // 0.0 - 1.0
  style: number;           // 0.0 - 1.0 (v2 models only)
  use_speaker_boost: boolean;
}
```

### Stability

Controls consistency and expressiveness.

```
Low Stability (0.0-0.3)
├── More emotional variation
├── Dynamic intonation
├── Natural speech patterns
├── Risk: May sound unstable
└── Best for: Drama, storytelling, social

Medium Stability (0.4-0.6)
├── Balanced expression
├── Natural but controlled
├── Predictable output
└── Best for: Most content types

High Stability (0.7-1.0)
├── Consistent delivery
├── Minimal variation
├── Robotic at extremes
└── Best for: Corporate, technical, instructions
```

### Similarity Boost

Controls fidelity to original voice.

```
Low Similarity (0.0-0.4)
├── More creative freedom
├── May drift from original
├── Unique interpretations
└── Rarely recommended

Medium Similarity (0.5-0.7)
├── Recognizable voice
├── Some artistic freedom
└── Good for most uses

High Similarity (0.8-1.0)
├── Closest to original
├── Maintains voice identity
├── Most predictable
└── Best for brand consistency
```

### Style (Multilingual v2 Only)

Controls expressiveness exaggeration.

```
No Style (0.0)
├── Neutral reading
├── Even delivery
└── Best for: Instructions, technical

Light Style (0.1-0.3)
├── Subtle emphasis
├── Natural pacing
└── Best for: Tutorials, demos

Medium Style (0.4-0.6)
├── Noticeable expression
├── Engaging delivery
└── Best for: Marketing, social

Heavy Style (0.7-1.0)
├── Maximum expression
├── Dramatic delivery
├── Risk: Over-the-top
└── Best for: Entertainment, trailers
```

### Speaker Boost

Enhances clarity and presence.

```
Enabled (true) - Recommended
├── Clearer consonants
├── Better presence
├── Slight processing
└── Best for: Most content

Disabled (false)
├── Raw voice output
├── No enhancement
└── Best for: Specific artistic needs
```

### Multi-Speaker Implementation

```typescript
interface DialogueLine {
  speaker: string;
  text: string;
  voiceId: string;
  settings: VoiceSettings;
}

async function generateDialogue(
  lines: DialogueLine[]
): Promise<{ speaker: string; audio: Buffer }[]> {
  const results: { speaker: string; audio: Buffer }[] = [];

  for (const line of lines) {
    const audio = await generateSpeech(
      line.text,
      line.voiceId,
      line.settings
    );

    results.push({
      speaker: line.speaker,
      audio,
    });

    // Small delay to avoid rate limits
    await new Promise((r) => setTimeout(r, 100));
  }

  return results;
}

// Usage
const dialogue: DialogueLine[] = [
  {
    speaker: "narrator",
    text: "In this demo, we'll explore the new features.",
    voiceId: "21m00Tcm4TlvDq8ikWAM",
    settings: VOICE_PRESETS.professional,
  },
  {
    speaker: "user",
    text: "How do I get started?",
    voiceId: "ErXwobaYiN019PkySvjV",
    settings: VOICE_PRESETS.conversational,
  },
  {
    speaker: "assistant",
    text: "Just run this simple command.",
    voiceId: "pNInz6obpgDQGcFmaJgB",
    settings: VOICE_PRESETS.professional,
  },
];
```

## Voice Testing

### A/B Testing Framework

```typescript
interface VoiceTest {
  name: string;
  voiceId: string;
  settings: VoiceSettings;
}

async function runVoiceTest(
  text: string,
  variants: VoiceTest[]
): Promise<Map<string, Buffer>> {
  const results = new Map<string, Buffer>();

  for (const variant of variants) {
    const audio = await generateSpeech(
      text,
      variant.voiceId,
      variant.settings
    );
    results.set(variant.name, audio);
  }

  return results;
}

// Example: Test voices for a product demo
const variants: VoiceTest[] = [
  {
    name: "professional-adam",
    voiceId: "pNInz6obpgDQGcFmaJgB",
    settings: VOICE_PRESETS.professional,
  },
  {
    name: "friendly-rachel",
    voiceId: "21m00Tcm4TlvDq8ikWAM",
    settings: VOICE_PRESETS.conversational,
  },
  {
    name: "energetic-antoni",
    voiceId: "ErXwobaYiN019PkySvjV",
    settings: VOICE_PRESETS.marketing,
  },
];

const testResults = await runVoiceTest(
  "Welcome to our product. Let me show you what's possible.",
  variants
);
```
