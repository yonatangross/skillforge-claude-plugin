# Music Matching Guide for Tech Demo Videos

Detailed guide for selecting the perfect music track to match your video content, mood, and pacing.

## Content-to-Music Decision Framework

### Step 1: Identify Video Purpose

| Purpose | Primary Emotion | Musical Direction |
|---------|-----------------|-------------------|
| Educate | Clarity, Focus | Minimal, unobtrusive |
| Impress | Awe, Excitement | Building, dynamic |
| Explain | Understanding | Supportive, steady |
| Convince | Trust, Confidence | Professional, resolving |
| Entertain | Engagement | Energetic, memorable |

### Step 2: Map Content Sections

```
Intro (0-10s)        → Hook: Attention-grabbing opening
Problem (10-30s)     → Tension: Minor key, building
Solution (30-90s)    → Resolution: Major key, hopeful
Demo (90-180s)       → Engagement: Matching action pace
Conclusion (180-210s) → Triumph: Satisfying resolution
CTA (210-240s)       → Energy: Motivating, forward-moving
```

## BPM Mapping to Visual Pacing

### Matching Cuts to Beats

```
BPM 60:  1 beat per second  → Slow, contemplative edits
BPM 90:  1.5 beats/second   → Standard pacing
BPM 120: 2 beats per second → Quick cuts, dynamic
BPM 150: 2.5 beats/second   → Rapid montage
```

### Scene Duration by BPM

| BPM | Beats per Bar | Bar Duration | Natural Cut Points |
|-----|---------------|--------------|-------------------|
| 70 | 4 | 3.43s | Every 2-4 bars |
| 90 | 4 | 2.67s | Every 2-4 bars |
| 110 | 4 | 2.18s | Every 4-8 bars |
| 130 | 4 | 1.85s | Every 4-8 bars |

### Tempo Changes

**Speeding Up:**
- Building excitement
- Approaching climax
- Increasing complexity

**Slowing Down:**
- Emphasizing importance
- Allowing comprehension
- Preparing for transition

## Mood Progression Patterns

### The Arc Pattern
```
Start: Neutral/Curious (90 BPM, mid energy)
Build: Increasing interest (100 BPM, rising)
Peak: Maximum engagement (110 BPM, high energy)
Resolve: Satisfaction (100 BPM, positive)
End: Call to action (105 BPM, forward momentum)
```

### The Reveal Pattern
```
Start: Mystery/Anticipation (85 BPM, minor key)
Build: Growing tension (90 BPM, suspense)
Reveal: The moment (95 BPM, key change to major)
Celebrate: Impact (100 BPM, triumphant)
Settle: New normal (95 BPM, confident)
```

### The Journey Pattern
```
Start: Beginning point (80 BPM, simple)
Progress: Adding layers (90 BPM, building)
Challenges: Complexity (95 BPM, interesting)
Overcome: Solutions (100 BPM, resolving)
Arrival: Destination (95 BPM, satisfied)
```

## Track Structure Alignment

### Typical Track Structure
```
Intro (8-16 bars)      → Video intro, logo, title
Verse (16 bars)        → Problem/context setup
Pre-chorus (8 bars)    → Building anticipation
Chorus (16 bars)       → Main demo/feature showcase
Verse 2 (16 bars)      → Additional features
Chorus 2 (16 bars)     → Key benefits
Bridge (8 bars)        → Testimonial/results
Final Chorus (16 bars) → Recap, call to action
Outro (8-16 bars)      → End screen, subscribe
```

### Editing to Music Structure

1. **Mark musical sections** in your timeline
2. **Align major transitions** with section changes
3. **Use drop points** for reveals
4. **Place key messages** on downbeats
5. **End scenes** on phrase endings

## Quality Checklist

Before finalizing music selection:

- [ ] Tempo matches video pacing
- [ ] Mood aligns with message
- [ ] Energy level appropriate
- [ ] No distracting elements
- [ ] Builds at right moments
- [ ] Resolution feels satisfying
- [ ] License covers usage
- [ ] Mix quality is professional
- [ ] Length matches video (or can loop)
- [ ] Works with narration/SFX

## Common Mistakes to Avoid

1. **Overpowering music** - Should support, not dominate
2. **Wrong tempo** - Mismatched pacing feels jarring
3. **Inappropriate mood** - Upbeat music with serious content
4. **Too complex** - Busy arrangements distract
5. **Poor quality** - Low-quality audio undermines credibility
6. **Mismatched cuts** - Edits that fight the rhythm
7. **Abrupt endings** - Music stopping without fade
8. **Genre mismatch** - Style that doesn't fit brand
