## Phase 2: Content Creation

### 2.1 Script Development

```
[ ] Hook written and tested
    └─▶ Creates curiosity in < 3 seconds
    └─▶ Speaks to audience pain/desire
    └─▶ Read aloud and timed

[ ] Full script draft completed
    └─▶ AIDA structure followed
    └─▶ Word count appropriate for duration
    └─▶ Technical accuracy verified

[ ] Script timing validated
    └─▶ Read aloud at natural pace
    └─▶ Fits within target duration
    └─▶ Breathing room for visuals

[ ] Call-to-action finalized
    └─▶ Single, clear action
    └─▶ Specific command/URL
    └─▶ Easy to remember/copy
```

### 2.2 Storyboard Creation

```
[ ] Scene breakdown completed
    └─▶ Each scene documented with template
    └─▶ Timing allocated per AIDA phases
    └─▶ Transitions defined

[ ] Shot list finalized
    └─▶ Every shot numbered and described
    └─▶ Duration specified for each
    └─▶ Shot type identified

[ ] Visual style defined
    └─▶ Color palette documented
    └─▶ Typography chosen
    └─▶ Animation style decided
    └─▶ Reference examples collected

[ ] B-roll requirements listed
    └─▶ All needed footage identified
    └─▶ Priority marked (essential vs nice-to-have)
    └─▶ Capture sessions scheduled
```

### 2.3 Script Checklist

```
ATTENTION PHASE:
[ ] Hook is under 5 seconds
[ ] Creates immediate curiosity
[ ] No logo animation as opener
[ ] Pattern interrupt present

INTEREST PHASE:
[ ] Problem clearly articulated
[ ] Solution introduced smoothly
[ ] 3 or fewer features highlighted
[ ] Benefits stated, not just features

DESIRE PHASE:
[ ] Proof elements included
[ ] Emotional connection built
[ ] Transformation shown
[ ] Differentiation clear

ACTION PHASE:
[ ] Single primary CTA
[ ] Specific next step
[ ] Low friction (time/effort stated)
[ ] Brand closure included
```

## Quick Reference: Common Issues

```
┌─────────────────────────────────────────────────────────────────┐
│                    COMMON PRE-PROD ISSUES                        │
├──────────────────────────────────┬──────────────────────────────┤
│  Issue                           │  Prevention                  │
├──────────────────────────────────┼──────────────────────────────┤
│  Script too long for duration    │  Time script read-aloud      │
│  Screen capture has errors       │  Test all commands first     │
│  Missing b-roll footage          │  Complete shot list early    │
│  Inconsistent visual style       │  Create style guide first    │
│  Audio doesn't match video       │  Sync timing in storyboard   │
│  Last-minute changes             │  Get sign-off at each phase  │
│  Asset resolution too low        │  Specify requirements upfront│
│  Missing fonts/licenses          │  Collect brand assets early  │
│  Scope creep                     │  Define goals clearly first  │
│  Timeline slippage               │  Build in buffer time        │
└──────────────────────────────────┴──────────────────────────────┘
```
