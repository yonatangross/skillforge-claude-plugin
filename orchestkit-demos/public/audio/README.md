## Required: Background Music
Download a royalty-free ambient/tech track and save as `ambient-tech.mp3`

### Recommended Sources (Free):
1. **Pixabay** (no attribution required)
   - https://pixabay.com/music/search/ambient%20tech/
   - Recommended: "Technology Corporate" or "Ambient Tech Background"

2. **Mixkit** (free license)
   - https://mixkit.co/free-stock-music/tag/tech/

3. **YouTube Audio Library** (if publishing to YouTube)
   - https://studio.youtube.com/channel/UC/music

### Requirements:
- Duration: ~15-20 seconds (will loop/fade)
- Style: Subtle, ambient, electronic
- Tempo: Slow to medium (60-90 BPM)
- No vocals

### Quick Download (Pixabay example):
```bash
# Example - replace URL with actual track
```
